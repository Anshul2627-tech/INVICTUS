# INVICTUS 🔥
*Fight in Hell. Survive the Waves. Stay Unconquered.*

A third-person combat game built in Unity where you fight waves of enemies in a hellish arena.

---

## 🎮 Gameplay
- Fight endless waves of demons
- Each wave gets harder — more enemies spawn
- Punch enemies to defeat them
- Survive as long as possible

## ⚔️ Controls
| Action | Key |
|--------|-----|
| Move | WASD |
| Look | Mouse |
| Punch | Left Click |

## 🛠️ Built With
- Unity 6 (URP)
- C#
- Unity NavMesh AI
- Inferno VFX (Mirza Beig)
- Galaxy Fire Skybox

## 📦 How to Run
1. Clone the repo
2. Open in Unity 6
3. Open `Scenes/Invictus` scene
4. Press Play
